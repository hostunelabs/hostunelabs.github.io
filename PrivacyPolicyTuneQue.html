Privacy Policy

Last updated: 28 March 2026

This Privacy Policy describes how the Android TV Token Display System ("Application", "App") handles information when used in hospitals, clinics, and other healthcare facilities.

We are committed to protecting user privacy and ensuring transparency regarding how data is handled within the Application.

1. Purpose of the Application

The Android TV Token Display System is designed to display queue tokens, announcements, and related information on Android TV devices within hospitals or clinics. The App functions as a display-only system and does not allow patient interaction.

2. Personal Information

The Application does not collect, store, or request any personal information from users.

  • No patient data is entered on the TV device
  • No user registration is required
  • No phone number, email, or personal details are collected
  • No biometric data is collected

All displayed information is received from a secure hospital-controlled server and is shown only for display purposes.

3. Device & Network Information

For operational purposes, the Application may:

  • Connect to a hospital-controlled server
  • Use local device network access to retrieve token data
  • Store temporary cached display data locally for performance

The App does not track users and does not perform device fingerprinting. No location tracking is performed.

4. Use of Information

Any data displayed in the Application is:

  • Provided by the hospital server
  • Used only for real-time token display
  • Not stored permanently on the device
  • Not transmitted to any third-party services

The Application does not use data for:

  • Advertising
  • Analytics
  • Marketing
  • Profiling

5. Third-Party Services

This Application does not use any third-party services, including analytics platforms, advertising SDKs, or social media integrations. The App operates only with the configured backend server.

6. Data Sharing

The Application does not share any data with:

  • Third-party companies
  • Advertising networks
  • Social media platforms
  • External analytics providers

All information remains within the hospital's own infrastructure.

7. Data Storage & Retention

The Application may store temporary cached data locally for display performance. This data is automatically overwritten and is not permanently retained. No personal data is stored.

8. Children's Privacy

This Application is not intended for use by children. The App does not collect any personal information from anyone, including children.

9. Security

The Application communicates only with configured servers. Reasonable technical measures are used to protect data during transmission. Since the App functions as a display system, minimal data processing occurs on the device.

10. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Changes will be reflected by updating the "Last updated" date.
